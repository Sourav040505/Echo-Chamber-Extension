# Echo Chamber Breaker

A Chrome extension that detects ideological bias in real-time and helps users consume more balanced content online.

## What it does

- Scans the content you're reading and scores it for ideological lean using OpenAI API and BERT models
- Shows a Perspective Meter overlay so you can see how one-sided your feed is
- Awards gamified badges when you explore content outside your usual bubble
- Works silently in the background — no censorship, just awareness

## Tech Stack

React.js · Chrome Extension APIs · Firebase · OpenAI API · BERT · Perspective API

## Getting Started

1. Clone the repo
```bash
   git clone https://github.com/yourusername/echo-chamber-breaker.git
   cd echo-chamber-breaker
```
2. Install dependencies
```bash
   npm install
```
3. Add your API keys in a `.env` file
